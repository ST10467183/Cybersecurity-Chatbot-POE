# Cybersecurity Awareness Chatbot

A console-based cybersecurity chatbot for PROG6221 POE Part 1.

# Features
- 🎤 Voice greeting on startup
- 🎨 ASCII art logo with colored console UI
- 👤 Personalized responses using user's name
- 🔐 Cybersecurity tips for:
  - Password safety
  - Phishing awareness
  - Safe browsing
  - Scam prevention
  - Privacy protection
  - Two-factor authentication (2FA)

## How to Run
1. Open `CybersecurityChatbot.sln` in Visual Studio 2022.
2. Press `F5` to run.
